079 | Основы программировани

Description

A gentle introduction to corpus processing and programming for computational linguists.

Course objectives

By the end of this course you should be:
  • familiar with the syntax of the Python programming language
  • able to navigate the filesystem using the UNIX shell
  • capable of efficiently using online resources to guide your own learning
  • able to use Python to process corpora from the web
  • able to use Python to implement a simple NLP pipeline from scratch

Lectures and practicals

Week 1

  • Install Python
  • Where to get help ? Google, Stackoverflow, IRC, docs.python.org
  • Pipelines in UNIX
  • Processing Wikipedia dumps
  • Basic structure of a program
  • Programming patterns for input/output (stdin, stdout)
Practical: Download and extract a Wikipedia dump for a language of your choice. Homework: Use a standard UNIX utility to count the number of lines, words and characters in your corpus.

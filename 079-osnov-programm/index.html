079 | Основы программировани

Description

A gentle introduction to corpus processing and programming for computational linguists. The course is structured in two parts, the first six weeks is familiarisation with programming in Python and the UNIX shell, the second six weeks is implementing a program of your choice to solve a personal need related to linguistics or natural-language processing. Every week we will do a short lecture where I will introduce some concepts, and then the remainder of the course will be practical work — that is, programming.

Course objectives

By the end of this course you should be:
  • familiar with the syntax of the Python programming language
  • able to navigate the filesystem using the UNIX shell
  • capable of efficiently using online resources to guide your own learning
  • able to use Python to process corpora from the web
  • able to use Python to implement a simple NLP pipeline from scratch

Lectures and practicals

Week 1

Preparation: Get some sleep, prepare for excitement.

  • Install Python
  • Where to get help ? Google, Stackoverflow, IRC, docs.python.org
  • Pipelines in UNIX
  • Processing Wikipedia dumps
  • Basic structure of a program
  • Programming patterns for input/output (stdin, stdout)

Practical: Download and extract a Wikipedia dump for a language of your choice.

Homework: Use a standard UNIX utility to count the number of lines, words and characters in your corpus.

Week 2

Preparation: Read the informal introduction to Python, here.

  • Continue with basic structure of programs
  • Variables
  • Arithmetic in Python
  • Loops and flow control
  • String processing

Practical: Write a simple sentence segmenter in Python for the corpus you have created.

Homework: Write your own program in Python to count the number of lines, words and characters in your corpus. Compare the results with the output of the UNIX tool.

Week 3

Preparation:

  • Character encoding and special characters
  • Lists
  • Formatting output using print()

Practical: Write a basic tokeniser that takes the output of the sentence segmenter and outputs the tokens one per line with token ids.

Homework: Improve the segmenter and tokeniser to take into account more cases. Annotate ten sentences from your corpus for part-of-speech tags.

  • File input/output
  • Functions

Practical: Write a program to either transliterate or transcribe the sentences in your corpus

Homework: Annotate 20 sentences from your corpus for part-of-speech

Preparation:

  • Type conversion
  • Sorting lists and dicts

Practical: Write a program to calculate a unigram language model from your tagged corpus
